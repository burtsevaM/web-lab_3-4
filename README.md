## Содержание проекта
- index.html
- store.html
- cart.html
- about.html
- styles.css
- script.js

## Инструкция для тестирования
Откройте файл `index.html` в браузере.

## Pull Request для одногруппника
1. Форкни репозиторий
2. Клонируй свой форк:
   git clone https://github.com/ТВОЙ_ЛОГИН/ИМЯ_ПРОЕКТА
3. Создай ветку:
   git checkout -b groupmate
4. Добавьте изменения и отправьте:
   git add .
   git commit -m "Описание изменений"
   git push origin groupmate
5. Создайте Pull Request на GitHub.

6. В конце кратко опиши что было сделано и почему
